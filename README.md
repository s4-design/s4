Кодо-ориентированная параметрическая система построения пользовательских интерфейсов. Основана на единой шкале относительных размеров, где визуальные характеристики интерфейса определяются набором взаимосвязанных правил и наследуемых зависимостей.

Система строится от примитивов к композициям, а главным объектом управления является **масштаб среды**, а не отдельный компонент.

<br>

---

5 архитектурных инвариантов:

1. **Три слоя `@layer`** — elements, themes, utilities — с фиксированным порядком каскада.
2. **Детерминированные формулы классов** — любой класс парсится в CSS-правило без контекста.
3. **Размерная шкала `--size--*`** — 33 именованных ступени (шаг ¼).
4. **Device-специфичная загрузка** — каждый `{device × orientation}` имеет свой CSS-файл.
5. **Темы** — `@scope ([theme=...])` с раздельной сборкой для каждого устройства.
