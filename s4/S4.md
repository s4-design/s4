# Дизайн-система S4

## Подключение

```html
<script src="/s4/js/s4.min.js"></script>
<script>S4()</script>
```

JS-фреймворк (Svelte, Vue и т.п.) подключается отдельно. S4 через `S4()` автоматически загружает CSS под текущее устройство и ориентацию.

Тема устанавливается автоматически: `S4()` определяет `prefers-color-scheme` и ставит атрибут `[theme=dark]` или `[theme=light]` на `<html>`. Если нужна кастомная тема — установите `[theme="..."]` на `<html>` до вызова `S4()`.

## Устройство и ориентация

Префикс класса = устройство + ориентация:

| Устройство | Ориентация |
|---|---|
| `d_` — десктоп | `l_` — альбомная |
| `m_` — мобильное | `p_` — портретная |
| `t_` — планшет | |

Примеры префиксов: `d_l_` (десктоп, альбомная), `m_p_` (мобильное, портретная), `t_p_` (планшет, портретная).

Пример класса: `d_l_display--flex` — десктоп-альбомная, `m_p_display--none` — мобильное-портретная.

## Утилитарные классы — 3 формулы

### Формула 1 — адаптивная

```
.{prefix}_{property}--{modifier}
```

Стиль привязан к устройству и ориентации.

| Класс | CSS |
|---|---|
| `d_l_display--flex` | `display: flex` |
| `m_p_position--sticky` | `position: sticky` |
| `d_l_align-content--space-around` | `align-content: space-around` |

С псевдоклассом: `.d_l_background-color--transparent\:hover:hover`

С псевдоэлементом: `.d_l_display--none\:\:before::before`

### Формула 2 — единая

```
.{property}--{modifier}
```

Стиль един для всех устройств и ориентаций.

| Класс | CSS |
|---|---|
| `.border--collapse` | `border-collapse: collapse` |
| `.box-sizing--border-box` | `box-sizing: border-box` |

### Формула 3 — переменная

```
.{property}
```

Подключает CSS-свойство к цепочке CSS-переменных темы:

```css
property: var(--{prefix}_{property}, var(--{property}));
```

| Класс | Результат |
|---|---|
| `.color` | `color: var(--d_l_color, var(--color))` |
| `.background-color` | `background-color: var(--d_l_background-color, var(--background-color))` |

С псевдоклассом: `.color\:hover:hover` → `color: var(--d_l_color\:hover, var(--color\:hover))`

## Темы

`S4()` определяет тему через `prefers-color-scheme` и устанавливает атрибут `[theme=dark]` или `[theme=light]` на `<html>`. Переключение в рантайме отслеживается.

**Важно:** каждая тема собирается отдельно для каждого устройства — `[theme=dark]` на десктопе может отличаться от `[theme=dark]` на мобильном (разные значения и даже наборы CSS-переменных). Равенство между устройствами не гарантировано. Для device-независимого стиля используйте Formula 2 или явные значения.

Цветовые переменные (акцентные цвета):

| Переменная | Описание |
|---|---|
| `--{x}` | Основной цвет |
| `--{x}--light` | Светлый оттенок |
| `--{x}--dark` | Тёмный оттенок |
| `--{x}--mute` | Полупрозрачный |

Где `{x}` — `positive`, `negative`, `prime`, `second`, `success`, `danger`.

Базовые: `--white`, `--black`, `--white--01`…`--white--09`, `--black--01`…`--black--09`.

Остальные группы CSS-переменных:

| Группа | Примеры |
|---|---|
| `--size--*` | `--size--1`, `--size--025` |
| `--font-size--*` | `--font-size--md`, `--font-size--l1` |
| `--margin--*`, `--padding--*`, `--gap--*` | `--margin--s3`, `--padding--md` |
| `--border-radius--*` | `--border-radius--s1`, `--border-radius--md` |
| `--box-shadow--*` | `--box-shadow--s1` |

## Элементы

Пользовательские элементы и компоненты, стилизуемые через CSS-переменные:

- `<e-badge>`, `<e-icon>`, `<e-popover>`, `<e-message>`, `<e-truncate>`, `<e-group>`, `<e-line>`
- `<menu>`, `<button>`, `<input>`, `<select>`, `<details>`, и другие HTML-элементы

## Примеры

Базовый HTML — элементы стилизуются без обязательных классов:

```html
<button>Button</button>
<input type="text" placeholder="Input" />
<table>
    <tr>
        <th>Head</th>
        <th>Head</th>
    </tr>
    <tr>
        <td>Cell</td>
        <td>Cell</td>
    </tr>
</table>
```
